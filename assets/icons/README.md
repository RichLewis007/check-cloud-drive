# Icons Directory

**Author:** Rich Lewis - @RichLewis007

Place your SVG icon files here. The app will automatically load them based on the drive type.

## Supported Icon Names

- `googledrive.svg` - For Google Drive remotes
- `onedrive.svg` - For OneDrive remotes
- `dropbox.svg` - For Dropbox remotes
- `protondrive.svg` - For Proton Drive remotes
- `unknown.svg` - Fallback for unknown drive types

## Icon Requirements

- Format: SVG
- Recommended size: 32x32 pixels (will be scaled automatically)
- The app will fall back to colored placeholder icons if SVG files are not found
